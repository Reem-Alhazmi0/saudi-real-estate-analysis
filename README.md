# saudi-real-estate-analysis

KSA Investment Dashboard
An interactive dashboard analyzing 359 real estate investment sites across 110 cities in Saudi Arabia, designed to support data-driven decision-making.  

Key Metrics:
- Scope: 359 sites across 13 administrative regions.  
- Financials: Average price cap of 804,927 SAR.  
- Top Regions: Riyadh (99 sites), followed by the Eastern Province and Makkah.  
- Asset Types: Diverse portfolio including Land, Apartments, Villas, and Townhouses.  

Technical Stack
- Data Preparation: Power Query for data cleaning and transformation.  
- Data Modeling: Power Pivot for building relational models.  
- Visualization: Pivot Tables and interactive charts for actionable reporting.  

Strategic Value
This project transforms raw data into a strategic tool to:
- Optimize operational efficiency by balancing regional workloads.  
- Identify investment opportunities in high-growth secondary cities.  
- Provide clear visibility into the real estate market maturity.
